## Secure Regex Data Extraction System
This is a python-based system that extracts data from provided text using regular expression (regex) patterns
## Files in repo
- secure_extractor.py: Main program
- test_input.txt: Sample input data
- requirements.txt: Python dependencies
##Features 
- **5 Data Types**: Email, Phone, URL, HTML Tags, Credit Cards
- **Security**: Input validation, dangerous pattern detection, data masking
- **Validation**: Luhn algorithm for credit cards, pattern verification
- **Output**: JSON and readable formats with statistics
## How to run Program
	python secure_extractor.py
